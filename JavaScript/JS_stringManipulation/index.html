String Manipulation

To Find String Length

                Syntax: 
                    string.length 
                
                Example: 
                    let a = "This Is JavaScript";
                    console.log(a.length);

String Concatination

Used to join strings

                Syntax: 
                    str1.concat(str2...)
                
                Example: 
                    let str1 = "This Is JavaScript";
                    let str2 = "This Is Css";
                    let result = str1.concat(" " + str2);
                    console.log(result)

subString Fetching

Using substr() function

                Syntax: 
                    Main_string.substr(string_index, numberOfCharactersFromIndex);
                
                Example: 
                    let longStr = "This Is JavaScript, a scripting language";
                    let newStr = longStr.substr(8, 10);
                    console.log(newStr);

subString Fetching

Using substring() function

                Syntax: 
                    Main_string.substring(string_index, numberOfCharactersFromIndex);
                
                Example: 
                    let longString = "This Is JavaScript, a scripting language";
                    let newString = longString.substring(8, 18);
                    console.log(newStr);

To Find the Position Of a Character in a string

Using indexof() function

                Syntax: 
                    string.indexOf('subString/character');
                
                Example: 
                    let indexString = "This Is JavaScript and It Is good and easy to understand ";
                    console.log(indexString.indexOf("good"));

To Ignore first n occurance - Using indexof() function

                Syntax: 
                    string.indexOf('subString/character', indexToSkip);
                
                Example: 
                    let indexString = "This Is JavaScript and It Is good and easy to understand ";
                    console.log(indexString.indexOf("and", 25));

TO find the position in reverse order - Using lastIndexof() function

                Syntax: 
                    string.lastIndexOf('subString/character');
                
                Example: 
                    let indexString = "This Is JavaScript and It Is good and easy to understand ";
                    console.log(indexString.indexOf("JavaScript"));

To trim white space

trim white space from start and end of the string

                Syntax: 
                    string.trim();
                
                Example: 
                    let indexString = " This Is JavaScript and It Is good and easy to understand ";
                    console.log(indexString.trim());

trim white space from start of the string

                Syntax: 
                    string.trimStart();
                
                Example: 
                    let indexString = " This Is JavaScript and It Is good and easy to understand ";
                    console.log(indexString.trimStart());

trim white space from end of the string

                Syntax: 
                    string.trimEnd();
                
                Example: 
                    let indexString = " This Is JavaScript and It Is good and easy to understand ";
                    console.log(indexString.trimEnd());

Check If a subString Exists in the string or NO

Using includes() function (it returns boolean values) 

                Syntax: 
                    string.includes('sunString to search');
                
                Example:
