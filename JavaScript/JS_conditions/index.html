Conditions in JavaScript

simple if and if else

simple if : it is used to check the condition and if the condition is true the statement inside the if condition is executed

if else : it is used to check the condition and if the condition is true the statement inside the if condition is executed if its fasle the else part is executer

syntax - 
        for simple if: 
             if(argument){
                statement;
             }
             
        for if else:
             if(argument){
                statement;
             }
             else{
                statement;
             }

else if

else if : it is used to check multiple conditions and any number of else if's can be used

        syntax -
            if(argument){
                statement;
            }
            else if(argument){
                statement;
            }
            else{
                statement;
            }

Note: else part is optional

Nested if else

        if(argument){
            if(argument){
                statement;
            }
            else{
                statement;
            }
        }
        else{
            statement;

               (or)

            if(argument){
                statement;
            }
            else{
                statement;
            }
        }

Ternary Operator in JavaScript

        syntax: 
            condition ? statement1 : statement2;
            //condition      true       : false;

It checks the condition and if it's true it executes the first statement and if it's false it executes the second statement

Null is collesing Operator in JavaScript

If a variable is undefined this operator will show its alternating output which the programer givs.

        symbol to use: ??
