Loops in JavaScript

while loop: 

        syntax;
            while(argument or condition)
            {
                -----------
            }

            example:
            let counter = 1;
            while(counter <= 10)
            {
                console.log("charan while");
                counter++;
            }

do while loop: 

        syntax;
            do
            {
                -----------
            } while(argument or condition);

            example:
            let counter = 1;
            do
            { 
                console.log("charan do white");
                counter++;
            } while(counter <= 10);

for loop: 

        syntax;
            for(variable: condition: increment/ decrement)
            {
                -----------
            }

            example:
           
            for(let a = 1; a <= 5; a++)
            {
                console.log("charan for");
                counter++;
            }

Break, Continue

break : It breakes the loop i.e, it terminates the loop
Continue : It doesn't print the condition which is given to it. it just bypasses to that perticular condition
